# INCIT TEST BE

## Tech Stack Used
- ExpressJS (NodeJS)
- Typescript
- JWT
- Google OAuth
- Nodemailer (Sending email)
- Drizzle ORM
- Postgresql

## How to run
- Run `npm install` to install all dependencies
- Setup `.env` file for all credentials
- Run `npx drizzle-kit migrate` to run database migration
- To start in dev mode, run `npm run dev`
